# プレビュー画面とTinyMCEの表示スタイル統一

## 目的

[[doc-content-tinymce]]の編集領域（TinyMCE）で見えるリッチテキスト（見出し・箇条書き・表・リンクなど）の見た目（余白・文字サイズ・箇条書きの記号・表の罫線など）と、保存後にプレビュー画面（詳細ページ）へ表示されるHTMLの見た目を一致させる。

## 概要

- プレビュー画面とTinyMCEエディタの編集領域の両方から読み込む、共通のスタイルファイルを新設する。
- 見出し・段落・箇条書き・リンク・表・インラインコードなど、TinyMCEのツールバーで入力できる要素の見た目を、共通スタイルファイル1つで定義する。

詳細設計は[[../detail/doc-content-style]]を参照。
